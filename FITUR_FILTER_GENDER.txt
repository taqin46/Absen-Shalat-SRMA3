1. STRUKTUR DATA:
   const ALL_STUDENTS = {
     laki: [ {...}, {...}, ... ], // 45 siswa
     perempuan: [ {...}, {...}, ... ] // 37 siswa
   }

2. FILTER STATE:
   let currentFilter = 'semua'; // atau 'laki' atau 'perempuan'

3. FILTER FUNCTION:
   filterByGender(gender) {
     currentFilter = gender;
     loadStudentData(); // render ulang tabel
     updateFilterStatus(); // update status bar
   }

4. GET FILTERED DATA:
   getFilteredStudents(ALL_STUDENTS) {
     if (currentFilter === 'semua') return all 82
     if (currentFilter === 'laki') return 45
     if (currentFilter === 'perempuan') return 37
   }

5. RENDER TABLE:
   renderTable() menggunakan data terfilter
   updateTotalSiswa() update count
   updateFilterStatus() update indicator
